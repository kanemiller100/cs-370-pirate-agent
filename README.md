# CS 370 Portfolio Artifact: Pirate Intelligent Agent

## Project Overview
This repository contains my Jupyter Notebook for the pirate intelligent agent project from CS 370.

## Reflection

In this project, I created a pirate intelligent agent within a Jupyter Notebook based on the concepts of reinforcement learning and neural networks. The aim of this project was to create an agent that would effectively traverse a maze and find the treasure by learning from experiences rather than adhering to a set of instructions. This project demonstrates the application of machine learning to decision-making situations that are normally present within a maze.

The code was partially provided within the project starter package, which included the environment configuration, the maze representation, and additional auxiliary code used to run the agent within the environment. I independently developed the main logic for the intelligent agent, which involved implementing the deep Q-learning methodology, outlining the learning process, defining the actions performed by the agent, and fine-tuning the agent to improve performance while learning to attain the treasure.

This project relates to the broader field of computer science by demonstrating the role that computer scientists play within the broader scope of creating systems that address problems and improve performance by relying on data and computation. A computer scientist does not simply write code; rather, they address a problem, create a solution, test the outcome of the solution, and improve the solution to perform optimally while addressing the issue presented by the problem. This is significant because computer science is used to improve almost every industry within the world of business and technology.

Throughout the course, I have learned that solving a problem as a computer scientist requires breaking down a big problem into smaller components, analyzing the patterns, assessing the solutions, and refining the design based on the results. In this project, I was required to critically think about the process of how the agent learns, the components that affect the success or failure of the agent, and the impact of the changes to the model. These requirements necessitated the technical implementation of the project.

As a computer scientist, my ethical responsibilities to the end user and the organization involve developing systems that are accurate, reliable, and responsible. As a computer scientist working with intelligent systems, it is imperative to consider the process of decision-making, the efficiency of the system, and the possible consequences of a poorly designed system. Computer scientists have the responsibility to develop systems that are fair, transparent, and reliable, protecting the interests of the users and the organization. Through this project, I learned that intelligent systems should be developed with diligence since the quality of the process affects the quality of the results.

In conclusion, this project provided a platform to apply the concepts of reinforcement learning and the neural network. It also provided a deeper understanding of the intelligent agent. It is a testament to my growth in the field of artificial intelligence and problem-solving. It is a valuable addition to my computer science portfolio.
